Instruksi Penghapusan Data (Data Deletion Instructions)

Terakhir Diperbarui: 20 Januari 2026

Sesuai dengan kebijakan privasi dan perlindungan data, kami menyediakan mekanisme bagi pengguna untuk menghapus data pribadi mereka yang tersimpan di sistem kami melalui integrasi pihak ketiga (Rchmd ID).

Jika Anda ingin menghapus aktivitas atau data akun Anda yang terkait dengan aplikasi kami, Anda dapat mengikuti langkah-langkah di bawah ini:

1. Cara Menghapus Melalui Pengaturan Media Sosial (Rekomendasi)

Anda dapat memutus akses aplikasi secara langsung melalui platform penyedia:

  • Facebook (Rchmd ID):
    1. Pergi ke Pengaturan & Privasi akun Facebook Anda.
    2. Pilih Aplikasi dan Situs Web.
    3. Cari Rchmd ID.
    4. Klik tombol Hapus.
  • Google (Rchmd ID):
    1. Buka Keamanan Akun Google.
    2. Cari Rchmd ID pada daftar aplikasi pihak ketiga.
    3. Pilih Hapus Akses.
  • Twitter/X (Rchmd ID):
    1. Buka Pengaturan Akun X Anda.
    2. Pilih Keamanan dan Akses Akun > Aplikasi dan Sesi.
    3. Pilih Rchmd ID dan klik Revoke Access.

2. Cara Menghapus Data dari Database Kami

Setelah Anda memutus akses melalui langkah di atas, data seperti ID, Nama, dan Foto Profil mungkin masih ada di database internal kami. Untuk menghapusnya secara permanen:

  1. Kirimkan email ke: rochmadabdulazis@yahoo.com
  2. Gunakan subjek: Permintaan Penghapusan Data - [Nama Platform, misal: Facebook/Google/Twitter(X)]
  3. Di dalam isi email, lampirkan Nama Lengkap atau ID Pengguna yang Anda gunakan agar kami dapat memvalidasi data tersebut.
  4. Tim kami akan memproses penghapusan data Anda dalam waktu 48 jam dan akan mengirimkan konfirmasi setelah data berhasil dihapus secara permanen.

3. Apa Saja Data yang Dihapus?

Setelah proses penghapusan selesai:

  • ID Pengguna unik dari provider akan dihapus.
  • Nama Lengkap dan Foto Profil akan dibersihkan dari server kami.
  • Anda tidak akan bisa login kembali ke akun lama Anda kecuali Anda melakukan registrasi ulang.
